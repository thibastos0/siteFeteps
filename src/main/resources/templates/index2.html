Sobre o Projeto

Tecnologia IoT Avançada

Aplicação de tecnologias da Indústria 4.0 com Internet das Coisas (IoT) para monitoramento em tempo real da cadeia de suprimentos refrigerada.

Conformidade Garantida

Solução desenvolvida para atender às normas sanitárias da ANVISA e ABNT, garantindo máxima segurança alimentar.

Especializado em produtos lácteos como sorvetes e manteiga, nosso sistema utiliza sensores ESP32 com Wi-Fi para monitoramento contínuo, interface web intuitiva e alertas automáticos via Telegram.
